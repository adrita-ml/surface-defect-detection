Surface Defect Detection

📌 Overview
This project focuses on detecting surface defects using image processing and machine learning techniques. It simulates real-world industrial quality inspection systems.

🚀 Problem Statement
Manual inspection of surface defects is inefficient and prone to errors. The goal of this project is to automate defect detection using computer vision.

🧠 Approach
- Image preprocessing (grayscale, filtering, edge detection)
- Feature extraction from images
- Defect detection logic based on image patterns
- Visualization of detected defects

📊 Results
- Successfully identified surface defects from input images
- Demonstrated practical use of image processing techniques

🛠️ Tech Stack
- Python
- OpenCV
- NumPy

📈 Key Learnings
- Working with real-world image data
- Applying image processing techniques
- Understanding defect detection pipelines

▶️ How to Run
1. Install dependencies:
  pip install -r requirements.txt
2. Run the project:
  python src/detect_defect.py
